# lttit

lttit 是一个“鸟群操作系统”（Tit Cluster Operating System），具备两个核心特性：

1. 将多个分布式节点当作一个整体来使用  
2. 节点可在运行时动态修改自身逻辑，实现整体一致性

## 系统组件

lttit 由多个可独立移植的模块组成，包括：

- **RTOS 内核**：双调度器（RT/BE）、同步原语、定时器
- **文件系统（FS）**：轻量类 Unix FS
- **内存管理（mg）**：多种适配 MCU 的分配器
- **数据结构库（lib）**：红黑树、Radix 树、哈希表、链表等
- **网络协议栈（TCP/IP）**：简化版 TCP/IP
- **Shell 与文本编辑器**：交互式命令行与轻量编辑器

这些组件构成了 lttit 的基础运行环境。

### 核心能力

#### CSC：分布式协议栈
让多个 MCU 节点像“一台机器”一样协同工作，提供：
- 路由（CCNET）
- 可靠传输（SCP）
- 远程调用（CCRPC）

#### ccBPF：动态编程组件
一个 C 子集编译器 + BPF 虚拟机，用于在运行时动态修改节点逻辑，例如调度策略、防火墙规则、流控算法等。

CSC 负责“让节点组成整体”，ccBPF 负责“让整体改变自身”。
